# Realtime chat app: text, voice, image [in a single room]

Connect and chat

## usage
```bash
git clone https://github/malibacollab/chat && cd chat
```

```bash
npm install
```

```bash
node server.js
```

or you can use this: 

```bash
npm start
```

now open browswer and go to : ```http://localhost:3000/```
